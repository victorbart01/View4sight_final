# View4Sight, site vitrine

Site marketing de **View4Sight**, SaaS de visualisation de nuages de points 3D et BIM. Next.js 15 (App Router), React 19, bilingue EN/FR.

## Démarrage

```bash
npm install
cp .env.example .env.local   # puis remplir les valeurs
npm run dev                  # http://localhost:3000
```
